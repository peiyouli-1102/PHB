# AGENTS.md

## 项目目标
搭建个人主页与第一篇技术博客，纯静态页面，使用 HTML+CSS+JS，部署在 GitHub Pages。

## 技术约束
- 纯静态，无框架，无构建工具
- 使用 Git 管理版本
- 开发环境：Ubuntu 本地文件 + 浏览器预览

## 页面需求
1. 首页：
   - 导航栏、个人简介（姓名+兴趣方向）、项目/学习展示卡片、博客入口按钮
   - 响应式布局（≥手机屏幕）
2. 博客列表页：显示第一篇博客条目，可点击
3. 第一篇博客页：
   - 主题：关于数值分析课程的笔记
   - 内容包含一部分课程资料（以文件/附件的形式），个人理解，作业整理（以文件福建的形式），AI 可辅助润色和排版
   - 返回主页链接

## 设计规范
- 神色背景，学术风格，卡片布局，最大宽度 900px
- 导航栏固定顶部，使用语义化标签
- CSS class 命名简洁

## AI 协作规则
- 每次任务前先读此文件
- 分步实施，需用户确认再做下一步
- 每步完成后提示测试方法
